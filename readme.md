# Kalender Bali
Kalender Bali merupakan sebuah library yang dapat digunakan untuk menghitung wuku dan wewaran berdasarkan perhitungan kalender bali

# Feature
- [x] Wuku
- [x] Eka Wara
- [x] Dwi Wara
- [x] Tri Wara
- [x] Catur Wara
- [x] Panca Wara
- [x] Sad Wara
- [x] Sapta Wara
- [x] Asta Wara
- [x] Sanga Wara
- [x] Dasa Wara
